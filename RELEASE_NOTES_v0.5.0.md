# MacEverything v0.5.0

这版补 Quick Look 预览和结果显示配置，让搜索结果浏览更接近成熟文件搜索工具。

## 推荐下载

推荐下载：`MacEverything-v0.5.0.dmg`

安装方式：打开 DMG，把 `MacEverything.app` 拖到 `Applications`。

## 新增

- 新增 Quick Look 快速预览。
  - 右上角菜单可以预览所选文件。
  - 结果右键菜单新增“快速预览”。
  - 新增 `⌘Y` 快捷键。
  - 当结果列表聚焦时，可以按空格预览。
- 新增结果显示配置。
  - 可显示/隐藏路径。
  - 可显示/隐藏修改日期。
  - 可显示/隐藏大小。
  - 可显示/隐藏类型。
  - 支持恢复默认显示。
- `settings.plist` 兼容升级：旧版本的索引目录、排除目录、历史和过滤器不会丢。

## 使用方法

1. 选中一个搜索结果。
2. 按 `⌘Y` 或右键选择“快速预览”。
3. 点击右上角 `...` → “显示列”，可以调整结果行显示内容。

## 注意

当前 DMG 仍未 Apple 公证，首次打开可能需要右键打开。

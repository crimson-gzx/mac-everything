# MacEverything v0.4.0

这版继续补 Everything 日常使用体验：搜索历史和常用过滤器/书签。

## 推荐下载

推荐下载：`MacEverything-v0.4.0.dmg`

安装方式：打开 DMG，把 `MacEverything.app` 拖到 `Applications`。

## 新增

- 新增搜索历史。
  - 执行搜索或打开结果时会记录当前查询。
  - 右上角菜单可一键套用历史查询。
  - 支持清空历史。
  - 最多保留 30 条，自动去重。
- 新增常用过滤器/书签。
  - 内置：PDF、图片、视频、今天修改、大文件。
  - 可把当前搜索保存为自定义过滤器。
  - 可套用过滤器。
  - 可删除过滤器。
  - 可恢复默认过滤器。
- `settings.plist` 兼容升级：旧版本的索引目录和排除目录不会因为新增历史/过滤器字段丢失。

## 使用方法

1. 打开 MacEverything。
2. 点击右上角 `...` 菜单。
3. 在“过滤器”里选择内置过滤器，或把当前搜索保存为过滤器。
4. 在“历史”里快速恢复之前用过的搜索。

## 注意

当前 DMG 仍未 Apple 公证，首次打开可能需要右键打开。

## 单条推文版本

I built MacEverything — an open-source, native macOS file search app inspired by Windows Everything.

It supports SQLite/FTS indexing, FSEvents live updates, global hotkeys, Quick Look preview, saved filters, search history, and Everything-like query syntax.

GitHub: crimson-gzx/mac-everything

## 更产品化版本

Mac users deserve an Everything-like file search experience.

So I built MacEverything: a native SwiftUI file search app for macOS.

- SQLite / FTS index
- FSEvents live updates
- global hotkey
- Quick Look preview
- saved filters
- search history
- exclusion folders
- performance metrics

Open source, MIT licensed.

GitHub: crimson-gzx/mac-everything

### Tweet 3

Example queries:

```text
*.pdf
!temp
name:report
path:Desktop
size:>100mb
date:today
```

The goal is not to replace Spotlight, but to make filename search feel instant and predictable.

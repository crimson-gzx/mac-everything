# MacEverything 首发推广计划

## 目标

让第一批用户知道：这是一个 macOS 上的 Everything 风格开源文件搜索工具。

核心转化动作：

1. 去 GitHub 看项目。
2. 下载最新版 DMG。
3. 给 Star。
4. 反馈 bug / 功能建议。

## 小红书发布建议

### 发布时间

- 工作日：中午 12:00 - 13:30，晚上 20:00 - 23:00
- 周末：下午 15:00 - 18:00，晚上 20:00 - 23:30

### 内容结构

1. 封面强钩子：Mac 上终于有 Everything 平替了
2. 第一段讲痛点：从 Windows 换 Mac 后，找文件变慢
3. 第二段讲产品：我做了 MacEverything
4. 第三段列功能：SQLite/FTS、全局快捷键、Quick Look、过滤器
5. 第四段讲使用方式：GitHub 下载 DMG
6. 结尾引导：欢迎 Star / 提 issue

### 配图清单

1. 封面图：软件截图 + 大标题
2. 搜索结果图：展示搜索 `*.pdf`
3. 菜单图：展示过滤器、历史、显示列
4. 性能图：展示“搜索 xxms · FTS · 候选 xxx”
5. GitHub 图：展示 Release 下载 DMG

## 推荐正文

从 Windows 换到 Mac 后，我最不习惯的一件事：

找文件没有 Everything 那种“秒出结果”的感觉。

所以我做了一个开源小工具：**MacEverything**。

它是一个原生 macOS 文件搜索工具，灵感来自 Windows Everything。

现在支持：

- 文件名快速搜索
- SQLite / FTS 本地索引
- FSEvents 实时监听文件变化
- 全局快捷键呼出
- Quick Look 快速预览
- 搜索历史
- 常用过滤器
- 索引目录和排除目录管理
- 类 Everything 搜索语法

比如可以搜：

```text
*.pdf
!temp
name:report
path:Desktop
size:>100mb
date:today
```

我现在还在持续优化速度和体验，最新版本已经能显示搜索耗时、FTS 候选数量和索引耗时。

目前还是早期版本，暂时没有 Apple 公证，第一次打开可能需要右键打开。

项目开源在 GitHub，MIT 许可证。

如果你也想要一个 Mac 上的 Everything 风格搜索工具，可以去 GitHub 下最新版 DMG。

项目名：**MacEverything**

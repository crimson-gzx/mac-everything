# MacEverything v0.9.0

这版继续优化速度路线：新增真正的增量 SQLite 写入。

## 推荐下载

推荐下载：`MacEverything-v0.9.0.dmg`

安装方式：打开 DMG，把 `MacEverything.app` 拖到 `Applications`。

## 新增

- 文件新增或修改后，直接 upsert 到 SQLite。
- 文件删除或目录删除后，直接从 SQLite 删除该路径及子路径。
- SQLite FTS 表会跟随增量变化同步更新。
- 增量写入使用事务。
- 增量写入失败时，自动回退到原来的延迟整库保存。

## 为什么这一步重要

v0.8.0 已经让普通关键词可以先走 SQLite FTS5 候选搜索。

v0.9.0 解决的是索引更新成本：日常新增、删除、移动、修改少量文件时，不需要再依赖整库保存思路，而是直接更新数据库。

## 仍然需要改进的地方

- 文件夹新增时，目前只会记录目录本身；如果一次性复制大量文件夹，仍可能触发完整重建。
- 下一步应做扫描进度 UI 和更细的性能指标显示。
- 之后可以继续优化数据库侧排序、分页和更完整的 FTS 查询语法。

## 注意

如果你从旧版本升级，建议点一次“重建索引”，保证 SQLite 和 FTS 表完整。

当前 DMG 仍未 Apple 公证，首次打开可能需要右键打开。
